Function Reference: days

datatypes: D = days (X)

Fixed-time duration in days.

D = days (X) returns a duration array representing fixed-time duration days equivalent to the values in X, which must be a numeric array.

days is also available as a method for duration arrays, in which case it performs the opposite conversion.

See also: duration, years, hours, minutes, seconds, milliseconds, duration.days
